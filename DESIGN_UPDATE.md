# 🎨 Обновление дизайна главной страницы

## ✨ Что изменилось

### 🚀 Новый Hero-раздел
- **Градиентный фон** с анимированными фигурами
- **Крупный заголовок** с градиентным текстом
- **Статистика** в реальном времени (количество задач, категорий, участников)
- **Интерактивный блок кода** с подсветкой синтаксиса Python
- **Современные кнопки** с иконками

- **3 карточки** с описанием ключевых функций
- **Анимированные иконки** с градиентным фоном
- **Hover-эффекты** для улучшения UX

### 📊 Обновленный контент
- **Современные карточки категорий** с иконками
- **Улучшенный список задач** с цветными метками сложности
- **Стильный лидерборд** с аватарами и медалями
- **Призыв к действию** в конце страницы

### 🎨 Дизайн-система
- **Современная цветовая палитра** с градиентами
- **Плавные анимации** и переходы
- **Адаптивный дизайн** для всех устройств
- **Консистентные отступы** и типографика

## 🔧 Технические улучшения

### CSS
- Добавлены **CSS-переменные** для консистентности
- **Flexbox и Grid** для современной верстки
- **Keyframe-анимации** для интерактивности
- **Media queries** для адаптивности

### HTML
- **Семантическая разметка** с правильными тегами
- **Доступность** (alt-атрибуты, ARIA-роли)
- **Оптимизированная структура** для SEO

### Мобильные (до 768px)
