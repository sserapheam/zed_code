# Платформа программирования с PostgreSQL

## Быстрый старт 🚀

### 1. Установка PostgreSQL
- **Windows**: Скачайте с https://www.postgresql.org/download/windows/
- **Linux**: `sudo apt install postgresql postgresql-contrib`
- **macOS**: `brew install postgresql`

### 2. Создание базы данных
```bash
# Подключитесь к PostgreSQL
psql -U postgres

# Создайте базу данных
CREATE DATABASE coding_platform;
\q
```

### 3. Установка зависимостей
```bash
pip install -r requirements_postgres.txt
```

### 4. Запуск приложения
```bash
python run_postgres.py
```

## Настройка переменных окружения 🔧

Создайте файл `.env`:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=coding_platform
DB_USER=postgres
DB_PASSWORD=your_password
FLASK_SECRET=your-secret-key
```

## Преимущества PostgreSQL версии ⚡

1. **Масштабируемость** - поддержка тысяч пользователей
2. **Производительность** - оптимизированные индексы
3. **Надежность** - ACID транзакции
4. **Расширяемость** - возможность кластеризации
5. **Безопасность** - продвинутые права доступа

## Следующие шаги для масштабирования 🎯

1. **Кэширование**: Redis для популярных запросов
2. **Очереди**: Celery для асинхронного выполнения кода
3. **CDN**: Для статических файлов
4. **Балансировка**: Nginx для распределения нагрузки
5. **Мониторинг**: Prometheus + Grafana

## Поддержка 💬

При возникновении проблем:
1. Проверьте логи PostgreSQL
2. Убедитесь в правильности пароля
3. Проверьте настройки файрвола
4. Запустите `python test_postgres.py` для диагностики
